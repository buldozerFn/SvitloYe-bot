Можливості

Повний набір інструментів для моніторингу відключень

📡

Автоматичне оновлення

Завантаження JSON із GitHub-репозиторію кожні 20 хвилин з кешуванням та перевіркою змін.

📅

Розклад на завтра

Автоматична публікація графіка на наступний день, щойно він з'являється у джерелі даних.

🔔

Сповіщення

Попередження за 10 хвилин до зміни та миттєве повідомлення при зміні стану (вкл/викл).

🏷️

Три зони

Відображення білої ⬜ (світло є), сірої 🟧 (можливе відключення) та чорної ⬛ (відключення) зон.

🛡️

Обробка помилок

Стійкість до проблем мережі, таймаутів, невалідного JSON та недоступності GitHub.

🤖

Команди бота

Інтерактивні команди: /today, /tomorrow, /now, /status для ручного перегляду інформації.

💬 Команди бота

/start

Привітання та список усіх доступних команд

/today

Повний погодинний графік відключень на сьогодні

/tomorrow

Графік на завтра (якщо вже опубліковано)

/now

Поточний стан та найближчі 8 годин

/status

Діагностика: час оновлення, дати в кеші

Сповіщення

📥 Fetch Task

Кожні 20 хв завантажує JSON, перевіряє хеш MD5, оновлює кеш та шукає новий розклад на завтра.

⏰ Alert Task

Щохвилини порівнює поточний час із розкладом. Відправляє попередження за 10 хв та при зміні стану.

🤖 Command Handler

Обробляє команди користувачів через aiogram Router. Форматує та відправляє графіки.

🚀 Швидкий старт

Три кроки до запуску бота

1

Встановіть залежності

$ pip install -r requirements.txt
2

Налаштуйте конфігурацію

Відкрийте bot.py та замініть значення:

BOT_TOKEN = "ваш_токен_від_BotFather"
CHAT_ID = "ваш_chat_id"

💡 CHAT_ID можна дізнатися через @userinfobot або через API

3

Запустіть бота

$ python bot.py
